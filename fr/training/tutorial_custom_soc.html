Utiliser des données de sol personnalisées¶

  • Objectif: Apprenez à charger des données personnalisées sur le carbone organique du sol pour calculer le sous-indicateur de changement de carbone à l’aide de Trends.Earth.

  • Temps estimé d’achèvement: 20 minutes

  • Accès Internet: Non requis

Note

Télécharger cette page en format PDF pour une utilisation hors connexion

Chargement de données personnalisées sur le carbone organique du sol¶

Note

Cet outil suppose que les unités de la couche raster à importer sont Métriques Tonnes de carbone organique par hectare. Si votre couche est dans des unités différentes, veuillez effectuer les conversions nécessaires avant de l’utiliser dans Trends.Earth.

  1. Pour charger les données de carbone organique du sol, cliquez sur l’icône () dans la barre d’outils de Trends.Earth.

  1. Le menu Charger les données s’ouvre. Sélectionnez Carbone organique du sol dans la section Importer un jeu de données d’entrée personnalisé.

  1. Dans l’ensemble de données Charger un fichier de carbone organique personnalisé (SOC), utilisez le bouton radio pour sélectionner le format du fichier d’entrée (trame ou vecteur). Pour ce didacticiel, sélectionnez raster, car les données distribuées par UNCCD sont au format raster. Cliquez sur Parcourir pour accéder au fichier de carbone organique du sol que vous souhaitez importer.

  1. Utilisez la fenêtre Select input file pour accéder au fichier à importer, sélectionnez-le et cliquez sur Open.

  1. De retour à la fenêtre Charger un ensemble de données Custom Soil Organic Carbon (SOC), vous avez des options pour sélectionner le numéro de bande dans lequel les données de productivité sont stockées, dans le cas où votre fichier d’entrée est un raster multi-bande. Vous avez également la possibilité de modifier la résolution du fichier. Nous vous recommandons de les laisser comme paramètres par défaut sauf si vous avez des raisons valables de les changer.

  2. Définissez l’année de référence pour les données. Dans ce cas, nous supposerons que les données sur le carbone organique du sol remontent à 2000, mais si vous utilisez des données locales, assurez-vous d’attribuer la bonne année.

  3. Cliquez sur Parcourir en bas de la fenêtre pour sélectionner le fichier raster en sortie.

  1. Accédez au dossier dans lequel vous souhaitez enregistrer le fichier. Attribuez-lui un nom et cliquez sur Enregistrer.

  1. Retour à la Charger un ensemble de données Custom Soil Organic Carbon (SOC) cliquez sur OK pour que l’outil fonctionne.

  1. Une barre de progression apparaîtra montrant le pourcentage de la tâche terminée.

  1. Une fois le traitement terminé, l’ensemble de données sur le carbone organique du sol importé sera chargé dans QGIS.

Calcul du carbone organique du sol avec des données personnalisées¶

Une fois que vous avez importé un jeu de données personnalisé sur le carbone organique du sol, il est possible de calculer la dégradation du carbone organique du sol à partir de ces données. Pour ce faire, assurez-vous d’abord que les données de carbone organique du sol sont chargées dans QGIS (voir Chargement de données personnalisées sur le carbone organique du sol).

  1. Pour calculer la dégradation du carbone organique du sol à partir de données personnalisées, cliquez d’abord sur l’icône () dans la barre d’outils de Trends.Earth:

  1. Le menu « Calculer les indicateurs » s’ouvre. Sélectionnez « Carbone organique du sol » dans la section « Option 2: Utiliser des données personnalisées ».

  1. La fenêtre « Calculer le carbone organique du sol » s’ouvre. Cliquez sur le bouton radio à côté de « Ensemble de données de couverture terrestre personnalisé » et sélectionnez « Importer » pour importer un ensemble de données de couverture terrestre personnalisé ou « Charger existant » pour charger un jeu de données de couverture terrestre déjà traité dans Trends.Earth. Veillez à sélectionner à la fois une « couche initiale » et une « couche finale ». Voir le: tutoriel Utiliser des données de couverture terrestre personnalisées pour plus d’informations sur le chargement des jeux de données sur la couverture terrestre. Une fois que vous avez sélectionné les deux ensembles de données, cliquez sur suivant:

  1. Sur l’écran suivant, cochez la case « Données initiales du carbone organique initial du sol », puis utilisez les boutons « Importer » ou « Charger existant » pour importer la couche de carbone du sol (Chargement de données personnalisées sur le carbone organique du sol) ou charger un existant qui a déjà été calculé:

  1. Cliquez sur Suivant ». Maintenant, choisissez la zone dans laquelle vous souhaitez exécuter les calculs:

  1. Cliquez sur Suivant ». sur le dernier écran, entrez un nom de tâche ou des notes que vous souhaitez enregistrer (ceci est facultatif), puis cliquez sur « Calculer »:

  1. Une barre de progression apparaîtra sur votre écran. Ne quittez pas QGIS et n’éteignez pas votre ordinateur tant que le calcul n’est pas terminé.

  1. Une fois le calcul terminé, trois couches se chargeront sur votre carte: 1) la couche finale de carbone organique du sol, 2) la couche initiale de carbone organique du sol et 3) la couche de dégradation du carbone organique du sol:

  1. Par exemple, nous pouvons voir des zones de dégradation dans le carbone du sol autour de Kampala:

Reportez-vous au tutoriel: ref:tut_compute_sdg pour obtenir des instructions sur l’utilisation des données sur le carbone organique du sol importé pour calculer l’ODD final 15.3.1 après intégration avec la couverture terrestre et la productivité des terres.
