Changelog¶

Cette page répertorie l'historique des versions de | trends.earth |.

1.0.2¶

  • Première version de QGIS3 - de nombreux correctifs pour la mise à niveau vers Qt5 et QGIS3 API.

  • Mettez à niveau toutes les dépendances du plugin vers les dernières versions à partir de janvier 2020.

  • Correction de l'outil de téléchargement de données pour avoir des 1styles par défaut pour tous les jeux de données disponibles.

  • Commencez à passer aux frameworks QgsProcessing et QgsTask - actuellement, seul l'outil carbone est migré, mais tous les outils seront migrés avant la version 1.0./

  • Formatage de nettoyage de la feuille de calcul de sortie de l'outil carbone pour rendre la signification de chaque colonne plus claire.

  • Mettez à jour tous les scripts GEE pour utiliser la dernière version de l'API GEE (0.1.213).

  • Enregistrez plus de paramètres choisis dans les boîtes de dialogue d'outils entre les sessions QGIS.

  • Nettoyez le code de mise en mémoire tampon pour utiliser les projections Lambert Equal Area centrées sur les centroïdes de polygones pour la mise en mémoire tampon.

  • Déplacez la documentation dans le dossier docs à la racine du référentiel trends.earth.

  • Ajoutez plus de détails sur la façon de documenter comment contribuer au développement de Trends.Earth,

  • Nettoyez le référentiel en supprimant les fichiers de traductions compilés et en ajoutant ces types de fichiers à .gitignore.

  • Remplacez le nom du projet transifex par «tendances».

  • Diverses corrections de bugs et compatibilité.

`0,66 (20 juillet 2019) <https://github.com/ConservationInternational/trends.earth/releases/tag/0.66> `_¶

  • Limitez la superficie maximale des tâches à 10 000 000 km2, sauf pour les tâches en zone urbaine, qui est limitée à 10 000 km2.

  • Ajoutez une section d'arrière-plan sur SDG 11.3.1 et mettez à jour le didacticiel SDG 11.3.1.

  • Mettre à jour le code ODD 11.3.1 pour inclure 1998 dans la série (en interne pendant le calcul) afin de filtrer le bruit depuis le début de la série urbaine.

  • Correction des restrictions de date dans l'outil tout-en-un SDG 15.3.1 pour tenir compte à la fois de la disponibilité ESA et MODIS.

  • Ajoutez une section pour en savoir plus sur l'installation des versions de Github.

  • Mettez à jour et révisez les traductions espagnoles, mettez à jour les traductions de google pour d'autres langues.
