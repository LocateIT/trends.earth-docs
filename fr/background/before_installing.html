Avant d’installer l’outil¶

Avant d'installer la boîte à outils, QGIS version | qgisMinVersion | ou supérieur doit être installé sur votre ordinateur.

Télécharger QGIS¶

Pour installer le plugin, installez d'abord QGIS 3.10.3+ en suivant les étapes ci-dessous:

  1. Choisissez la version 32 ou 64 bits

    Vous avez la possibilité d'installer une version 32 bits ou 64 bits de QGIS. Pour connaître la version à installer, vérifiez le type de système d'exploitation dont vous disposez en suivant les instructions ci-dessous. Si vous ne savez pas de quoi vous avez besoin, essayez d'abord de télécharger la version 64 bits. Si cette version ne fonctionne pas correctement, désinstallez-la, puis installez la version 32 bits.

    • Windows 8 ou Windows 10

      • Dans l'écran &quot;Démarrer&quot;, tapez &quot;Ce PC&quot;.

      • Cliquez avec le bouton droit de la souris sur &quot;Ce PC&quot; (ou maintenez votre doigt dessus), puis cliquez sur &quot;Propriétés&quot;.

    • Windows 7 ou Vista

      • Ouvrez &quot;Système&quot; en cliquant sur le bouton &quot;Démarrer&quot;, en cliquant avec le bouton droit de la souris sur &quot;Ordinateur&quot;, puis en cliquant sur &quot;Propriétés&quot;.

      • Sous Système, vous pouvez voir le type de système.

    • Mac: Cliquez sur l'icône Apple en haut à gauche et sélectionnez &quot;À propos de ce Mac&quot;.

  2. Après avoir déterminé si vous avez besoin de la version 32 ou 64 bits, téléchargez le programme d'installation approprié:

    • Windows: `Téléchargez le programme d'installation de Windows à partir d'ici <https://qgis.org/en/site/forusers/download.html#windows> `_.

    • MacOS: `Téléchargez le programme d'installation de MacOS à partir d'ici <https://qgis.org/en/site/forusers/download.html#mac> `_.

    • Linux: `Téléchargez le programme d'installation Linux à partir d'ici, ou à partir du référentiel de votre distribution Linux <https://qgis.org/en/site/forusers/download.html#linux> `_.

Installer QGIS¶

Une fois le programme d'installation téléchargé à partir du site Web, il doit être exécuté (double-cliquez dessus). Sélectionnez les paramètres par défaut pour toutes les options.
